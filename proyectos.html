APLICACIÓN WEB

Durante la etapa 2 de Programación Web del Informatorio, aprendimos a programar en el lenguaje Python, siendo este orientado a objetos. Con mi grupo conformado por cuatro personas, trabajamos para asociarlo con el FrameWork Djando 3.0 a una Base de Datos creada en MySQL Server.

La aplicación web está basada en un Blog de videojuegos, donde existen usuarios de diferentes roles: administrador, estandar y visitante, cada uno con sus propias limitaciones.
